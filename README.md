# International Student Housing & Rental Life Knowledge Base

An LLM-powered personal knowledge base that helps international students understand
U.S. apartment rental concepts and processes.

## Project Structure

```
project2/
├── raw/                    # Raw source materials (manually collected guides)
│   ├── lease_terms_guide.md
│   ├── security_deposit_guide.md
│   ├── move_in_checklist.md
│   ├── move_out_guide.md
│   ├── utilities_guide.md
│   ├── renters_insurance_guide.md
│   ├── maintenance_requests_guide.md
│   ├── landlord_communication_guide.md
│   └── parking_rules_guide.md
├── wiki/                   # LLM-compiled knowledge base (generated by compile_wiki.py)
│   ├── index.md
│   ├── concepts/
│   ├── guides/
│   └── templates/
├── outputs/                # Q&A answers and lint reports (generated)
├── compile_wiki.py         # Step 2: Compile raw/ → wiki/
├── ask.py                  # Step 3: Q&A against the wiki
└── lint.py                 # Step 4: Health check the wiki
```

## Workflow

```
raw/ sources  →  compile_wiki.py  →  wiki/  →  ask.py (Q&A)
                                         └→  lint.py (health check)
```

## Setup

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY="your-key-here"
```

## Usage

### Step 1: Compile the wiki
```bash
python compile_wiki.py
```
Reads all 9 raw source files and uses Claude to generate 12 linked markdown articles in `wiki/`.

### Step 2: Ask questions
```bash
python ask.py "What should I do if my landlord doesn't return my security deposit?"
python ask.py "How do I set up electricity when I move in?"
python ask.py "What is normal wear and tear?"
python ask.py "Do I need renters insurance as an international student?"
python ask.py "What counts as an emergency maintenance request?"
```
Answers are streamed to the terminal and saved to `outputs/`.

### Step 3: Run a health check
```bash
python lint.py
```
Analyzes the wiki for missing links, content gaps, and improvement opportunities.
Report is saved to `outputs/lint_report.md`.

## Topics Covered

| Area | Coverage |
|------|----------|
| Lease terms | Types of leases, key clauses, renewal, early termination |
| Security deposit | Legal limits, deductions, normal wear and tear, dispute process |
| Move-in inspection | Room-by-room checklist, documentation, red flags |
| Move-out preparation | Timeline, cleaning checklist, written notice |
| Utilities | Electricity, gas, internet setup; cost estimates; international student tips |
| Renters insurance | Coverage, cost (~$10–20/month), how to get it (Lemonade etc.) |
| Maintenance requests | Emergency vs. routine, how to submit in writing, follow-up |
| Landlord communication | Written records, tone, email templates, certified mail |
| Parking rules | Permit types, towing process, what to ask before signing |

## Design Notes

- **Prompt caching** is enabled on all large context blocks to reduce API costs when
  asking multiple questions against the same wiki.
- **Streaming** is used in `ask.py` for a responsive terminal experience.
- The wiki uses `[[filename]]` cross-links compatible with Obsidian for browsing.
- All Q&A outputs are filed back into `outputs/` so sessions accumulate over time.
